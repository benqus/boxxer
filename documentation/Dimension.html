Dimension

new Dimension(arg)

Parameters:
Name Type Description
arg class is used and responsible for managing the rendered width or height of a Box instance. Each Box instance should have a width and a height Dimension instance which will calculate its own pixel/percent/weight values during the rendering process Every sizing calculation and info should be stored here instead of the Box class!
